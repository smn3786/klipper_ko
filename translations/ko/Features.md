# 기능

Klipper는 다음과 같은 여러 강력한 기능을 제공합니다:

* 고정밀 스테퍼 모터 제어. Klipper는 프린터
  동작을 계산할 때 애플리케이션 프로세서(저가의 Raspberry Pi와 같은)를
  활용합니다. 애플리케이션 프로세서는 각 스테퍼 모터를 언제 스텝할지
  결정하고, 해당 이벤트를 압축하여 마이크로컨트롤러로 전송한 다음,
  마이크로컨트롤러가 요청된 시간에 각 이벤트를
  실행합니다. 각 스테퍼 이벤트는 25마이크로초 이하의
  정밀도로 스케줄링됩니다. 소프트웨어는
  키네마틱 추정(Bresenham 알고리즘 등)을 사용하지 않고, 대신
  가속의 물리학과 기계 키네마틱스의 물리학을 기반으로
  정확한 스텝 시간을 계산합니다. 더 정밀한 스테퍼
  모터 제어는 더 조용하고 안정적인 프린터 작동을 제공합니다.

* 최고 수준의 성능. Klipper는 신형 및 구형
  마이크로컨트롤러 모두에서 높은 스테핑 속도를 달성할 수 있습니다. 심지어 구형 8비트
  마이크로컨트롤러도 초당 175,000스텝 이상을 얻을 수 있습니다.
  최신 마이크로컨트롤러에서는 초당 수백만 스텝이
  가능합니다. 높은 스테퍼 속도는 더 빠른 프린트 속도를 가능하게 합니다.
  스테퍼 이벤트 타이밍은 고속에서도 정밀하게 유지되어
  전체 안정성을 향상시킵니다.

* Klipper는 여러 마이크로컨트롤러를 사용하는 프린터를 지원합니다.
  예를 들어, 하나의 마이크로컨트롤러로 익스트루더를 제어하고,
  다른 하나로 프린터의 히터를 제어하며, 세 번째로
  프린터의 나머지 부분을 제어할 수 있습니다. Klipper 호스트 소프트웨어는
  마이크로컨트롤러 간의 클럭 드리프트를 고려하기 위해
  클럭 동기화를 구현합니다. 여러 마이크로컨트롤러를
  활성화하는 데 특별한 코드가 필요 없으며, 설정
  파일에 몇 줄만 추가하면 됩니다.

* 간단한 설정 파일을 통한 구성. 설정을 변경하기 위해
  마이크로컨트롤러를 다시 플래시할 필요가 없습니다. Klipper의 모든 설정은
  쉽게 편집할 수 있는 표준 설정 파일에 저장됩니다. 이를 통해
  하드웨어를 설정하고 유지 관리하기가 더 쉬워집니다.

* Klipper는 익스트루더 내부의
  압력 효과를 고려하는 메커니즘인 "부드러운 압력 전진"을 지원합니다. 이는
  익스트루더 "흘러내림"을 줄이고 프린트 모서리의 품질을 향상시킵니다. Klipper의
  구현은 순간적인 익스트루더 속도
  변경을 도입하지 않아 전체적인 안정성과 견고성을 향상시킵니다.

* Klipper는 진동이 프린트 품질에 미치는
  영향을 줄이기 위해 "입력 성형"을 지원합니다. 이는 프린트에서
  "링잉"("고스팅", "에코잉" 또는 "리플링"이라고도 함)을 줄이거나 제거할 수 있습니다.
  또한 높은 프린트 품질을 유지하면서
  더 빠른 프린팅 속도를 얻을 수 있습니다.

* Klipper는 간단한 키네마틱 방정식에서 정확한 스텝 시간을
  계산하기 위해 "반복 솔버"를 사용합니다. 이를 통해 Klipper를 새로운
  로봇 유형에 포팅하기가 더 쉬워지고, 복잡한
  키네마틱스에서도 타이밍을 정밀하게 유지합니다("라인 세그먼테이션"이 필요 없음).

* Klipper는 하드웨어에 구애받지 않습니다. 저수준
  전자 하드웨어와 무관하게 동일한 정밀한 타이밍을 얻을 수 있습니다.
  Klipper 마이크로컨트롤러 코드는 Klipper 호스트 소프트웨어가
  제공하는 스케줄을 충실히 따르도록 설계되었습니다(또는 불가능한 경우
  사용자에게 명확하게 알립니다). 이를 통해 사용 가능한 하드웨어를 더 쉽게 사용하고,
  새로운 하드웨어로 업그레이드하며, 하드웨어에 대한 신뢰를 가질 수 있습니다.

* 이식 가능한 코드. Klipper는 ARM, AVR, PRU 및 기타
  마이크로컨트롤러에서 작동합니다. 기존 "reprap" 스타일 프린터는
  하드웨어 수정 없이 Klipper를 실행할 수 있습니다 - Raspberry Pi만 추가하면 됩니다. Klipper의
  내부 코드 레이아웃은 다른
  마이크로컨트롤러 아키텍처도 더 쉽게 지원할 수 있도록 합니다.

* 더 간단한 코드. Klipper는 대부분의 코드에
  매우 높은 수준의 언어(Python)를 사용합니다. 키네마틱스 알고리즘, G-code 파싱,
  가열 및 서미스터 알고리즘 등은 모두 Python으로 작성되었습니다.
  이를 통해 새로운 기능을 개발하기가 더 쉬워집니다.

* 사용자 정의 프로그래밍 가능 매크로. 새로운 G-Code 명령을
  프린터 설정 파일에 정의할 수 있습니다(코드 변경이 필요 없음). 이러한
  명령은 프로그래밍 가능하여 프린터의 상태에 따라
  다른 동작을 생성할 수 있습니다.

* 내장 API 서버. 표준 G-Code 인터페이스 외에도
  Klipper는 풍부한 JSON 기반 애플리케이션 인터페이스를 지원합니다. 이를 통해
  프로그래머가 프린터를 세부적으로
  제어하는 외부 애플리케이션을 구축할 수 있습니다.

## 추가 기능

Klipper는 많은 표준 3D 프린터 기능을 지원합니다:

* 여러 웹 인터페이스 사용 가능. Mainsail, Fluidd,
  OctoPrint 등과 함께 작동합니다. 이를 통해
  일반 웹 브라우저를 사용하여 프린터를 제어할 수 있습니다. Klipper를 실행하는 동일한 Raspberry Pi가
  웹 인터페이스도 실행할 수 있습니다.

* 표준 G-Code 지원. 일반적인 "슬라이서"
  (SuperSlicer, Cura, PrusaSlicer 등)에서 생성하는 일반적인 G-code 명령이
  지원됩니다.

* 여러 익스트루더 지원. 공유 히터가 있는 익스트루더 및
  독립 캐리지(IDEX)의 익스트루더도 지원됩니다.

* 데카르트, 델타, corexy, corexz, hybrid-corexy,
  hybrid-corexz, deltesian, 회전 델타, 극좌표 및 케이블 윈치 스타일
  프린터를 지원합니다.

* 자동 베드 레벨링 지원. Klipper는 기본
  베드 기울기 감지 또는 전체 메쉬 베드 레벨링으로 구성할 수 있습니다. 베드 메쉬는
  프린트 크기에 맞게 사용자 정의할 수 있습니다(적응형 베드 메쉬). 베드가
  여러 Z 스테퍼를 사용하는 경우 Klipper는 Z 스테퍼를
  독립적으로 조작하여 레벨링할 수도 있습니다. BL-Touch 프로브 및
  서보 활성화 프로브를 포함한 대부분의 Z 높이 프로브가 지원됩니다. 프로브는
  축 비틀림 보상을 위해 캘리브레이션할 수 있습니다. "와전류
  프로브"를 사용하는 경우 빠른 베드 메쉬 스캐닝을 활용할 수 있습니다.

* 자동 델타 캘리브레이션 지원. 캘리브레이션 도구는
  기본 높이 캘리브레이션과 향상된 X 및 Y
  치수 캘리브레이션을 수행할 수 있습니다. 캘리브레이션은 Z 높이
  프로브 또는 수동 프로빙을 통해 수행할 수 있습니다.

* 런타임 "객체 제외" 지원. 구성된 경우 이 모듈은
  다중 부품 프린트에서 하나의 객체만 취소하는 것을
  용이하게 할 수 있습니다.

* 일반적인 온도 센서 지원(예: 일반 서미스터,
  AD595, AD597, AD849x, PT100, PT1000, MAX6675, MAX31855, MAX31856,
  MAX31865, BME280, HTU21D, DS18B20, AHT10, SHT3x 및 LM75). 사용자 정의
  서미스터 및 사용자 정의 아날로그 온도 센서도
  구성할 수 있습니다. 내부 마이크로컨트롤러
  온도 센서 및 Raspberry Pi의 내부 온도 센서를
  모니터링할 수 있습니다.

* 기본적으로 활성화된 기본 열 히터 보호.

* 표준 팬, 노즐 팬 및 온도 제어
  팬을 지원합니다. 프린터가 유휴 상태일 때 팬을 계속 실행할 필요가 없습니다.
  회전 속도계가 있는 팬에서 팬 속도를 모니터링할 수 있습니다.
  자동 팬 속도 업데이트를 위해 팬에 "수학 공식"을 할당할 수 있습니다.

* TMC2130, TMC2208/TMC2224,
  TMC2209, TMC2240, TMC2660 및 TMC5160 스테퍼 모터 드라이버의
  런타임 구성을 지원합니다. AD5206, DAC084S085, MCP4451, MCP4728, MCP4018 및 PWM 핀을 통한
  기존 스테퍼 드라이버의 전류 제어도 지원합니다.

* 프린터에 직접 연결된 일반 LCD 디스플레이를 지원합니다.
  기본 메뉴도 사용할 수 있습니다. 디스플레이 및 메뉴의 내용은
  설정 파일을 통해 완전히 사용자 정의할 수 있습니다.

* 일정한 가속 및 "미리보기" 지원. 모든 프린터 이동은
  정지 상태에서 순항 속도까지 점진적으로 가속한 다음
  정지 상태로 다시 감속합니다. 들어오는 G-Code
  이동 명령 스트림은 대기열에 추가되고 분석됩니다 - 유사한
  방향으로 이동 간의 가속이 최적화되어 프린트
  중단을 줄이고 전체 프린트 시간을 개선합니다.

* Klipper는 일반 엔드스톱 스위치의
  정확도를 향상시킬 수 있는 "스테퍼 위상 엔드스톱" 알고리즘을 구현합니다.
  적절하게 조정하면 프린트의 첫 번째 레이어 베드 접착력을 향상시킬 수 있습니다.

* 필라멘트 존재 센서, 필라멘트 모션 센서 및
  필라멘트 폭 센서를 지원합니다.

* adxl345,
  mpu9250, mpu6050, lis2dw12, lis3dh 및 icm20948 가속도계를 사용한
  가속도 측정 및 기록을 지원합니다.

* 프린터 진동 및 소음을 줄이기 위해
  짧은 "지그재그" 이동의 최고 속도를 제한하는 기능을 지원합니다.
  자세한 내용은 [키네마틱스](Kinematics.md) 문서를 참조하세요.

* 많은 일반 프린터용 샘플 설정 파일을 사용할 수 있습니다.
  목록은 [config 디렉토리](../config/)를 확인하세요.

Klipper를 시작하려면 [설치](Installation.md)
가이드를 읽어보세요.
